# TaradKU

A second-hand marketplace platform for students and staff of Kasetsart University, Sriracha Campus.

## 1. Project Overview
This project is developed for the course **Project Management and Digital Startup**, covering both project planning/management process and business model & product mockup design.

> Note: This is a Prototype/Mockup for academic purposes only. It is not a complete business plan for real-world operation.

## 2. Project Type
Mockup Web App — focused on demonstrating UI/UX and user flow with mock data. No real backend/database.

## 3. Tech Stack
| Layer | Technology |
|---|---|
| Markup | HTML5 |
| Style | CSS3, Bootstrap |
| Framework | Vue.js |
| Logic | Vue reactive state / JavaScript |
| Data | Mock JSON |
